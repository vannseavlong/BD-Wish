Share a personalized birthday surprise link with this URL format:

```
https://your-site.com/name=Seavlong&bd=Feb02
```

### URL Parameters

- `name` - The birthday person's name (e.g., `Seavlong`, `John`)
- `bd` - Birthday date (e.g., `Feb02`, `March15`, `Dec25`)

### Examples

1. **For Seavlong born on February 2:**
   ```
   /name=Seavlong&bd=Feb02
   ```

2. **For Maria born on December 25:**
   ```
   /name=Maria&bd=Dec25
   ```

3. **For Alex born on July 4:**
   ```
   /name=Alex&bd=July04
   ```
